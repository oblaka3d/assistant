# React UI для голосового ассистента

React приложение с тремя экранами и навигацией через свайпы.

## Структура

- **MainScreen** - главный экран с Unity персонажем
- **ChatScreen** - экран текстовой переписки (свайп влево)
- **MenuScreen** - экран меню и настроек (свайп вправо)

## Установка зависимостей

```bash
npm install
```

## Разработка

```bash
# Запуск dev сервера Vite
npm run dev

# Или из корня проекта
cd app/ui && npm run dev
```

## Сборка

```bash
# Из корня проекта
npm run build:ui

# Или
cd app/ui && npm run build
```

## Используемые библиотеки

- **React** - UI библиотека
- **Material-UI (MUI)** - компоненты интерфейса
- **react-chat-elements** - компоненты для чата
- **react-swipeable** - обработка свайпов

## Навигация

- **Свайп влево** - открыть чат (с главного экрана) или вернуться на главный (с меню)
- **Свайп вправо** - открыть меню (с главного экрана) или вернуться на главный (с чата)
