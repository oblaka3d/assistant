# @assistant/shared

Небольшой пакет с общими типами/утилитами. Сейчас содержит пример с поддерживаемыми локалями, но со временем сюда можно вынести:

- Общие интерфейсы API (`User`, `ApplicationSummary`, и т.д.)
- Хелперы форматирования дат/размеров, используемые в UI и backend
- Валидаторы/константы, которые должны совпадать между приложениями

Сборка: `npm run build --workspace @assistant/shared`.
