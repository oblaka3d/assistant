# Конфигурация Cursor Agent

Эта директория содержит конфигурацию для автоматической загрузки контекста в Cursor Agent.

## Файлы

- **context.md** - Инструкции о том, какие файлы загружать в контекст
- **commands/** - Пользовательские команды для Cursor

## Как это работает

1. **cursor.md** в корне проекта содержит ссылки на все файлы документации
2. **.cursorrules** указывает Cursor Agent автоматически загружать файлы по ссылкам из `cursor.md`
3. При упоминании `cursor.md` или вопросах о проекте, Cursor Agent должен:
   - Прочитать `cursor.md`
   - Автоматически загрузить все файлы из раздела "Структура документации"
   - Использовать их содержимое для ответов

## Структура документации

Все файлы документации находятся в `docs/`:

- `CURSOR_ARCHITECTURE.md` - Архитектура
- `CURSOR_COMMANDS.md` - Команды
- `CURSOR_TECHNICAL.md` - Технические детали
- `CURSOR_CONFIGURATION.md` - Конфигурации

## Использование

Просто упомяни `cursor.md` в запросе, и Cursor Agent автоматически загрузит всю необходимую документацию в контекст.
