# Контекстные файлы для Cursor Agent

Этот файл указывает, какие файлы документации должны быть автоматически загружены в контекст при работе с проектом.

## Обязательные файлы для загрузки

При начале работы с проектом или при запросах, требующих понимания архитектуры, всегда загружай следующие файлы:

1. **cursor.md** - Главный файл с навигацией и обзором
2. **docs/CURSOR_ARCHITECTURE.md** - Детальная архитектура проекта
3. **docs/CURSOR_COMMANDS.md** - Все команды и workflow
4. **docs/CURSOR_TECHNICAL.md** - Технические детали и решения
5. **docs/CURSOR_CONFIGURATION.md** - Конфигурации и настройки

## Как использовать

Когда пользователь задает вопрос о проекте:

1. Сначала прочитай `cursor.md` для общего понимания
2. Затем загрузи соответствующие файлы из `docs/` в зависимости от вопроса
3. Используй информацию из этих файлов для ответа

## Структура ссылок

Все ссылки в `cursor.md` ведут на файлы в `docs/` директории:

- `./docs/CURSOR_ARCHITECTURE.md`
- `./docs/CURSOR_COMMANDS.md`
- `./docs/CURSOR_TECHNICAL.md`
- `./docs/CURSOR_CONFIGURATION.md`

Эти файлы должны быть загружены автоматически при упоминании `cursor.md`.
